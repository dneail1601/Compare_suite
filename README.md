# Beyond Logic · Compare Suite

A minimalist, high-performance, and **zero-dependency** toolset for embedded developers, system administrators, and anyone who values efficient debugging.

The entire suite is **Offline-First**, extremely lightweight (< 100kB combined), and runs in any modern browser—making it perfect for deployment on Raspberry Pis or within isolated environments.

## The Suite

* **Hex Compare:** High-performance binary analysis using an intelligent 1KB chunk hash-scan. Perfect for identifying offsets and differences in large binary files.
* **JSON Tree Diff:** A tree-based comparison tool with "Lazy-Loading" logic. It loads only what is necessary, preserves RAM, and visualizes differences in nested configuration files instantly.
* **Text Diff:** Classic line-by-line comparison for configuration files.
* **Grep Tool:** Directory-wide search using regular expressions—runs locally in your browser with no server roundtrip.

## Why "Beyond Logic"?

* **Zero Dependencies:** No `node_modules`, no installation, no `npm`.
* **Hardware Agnostic:** Runs on any system (Raspberry Pi, Servers, Laptops) equipped with a web browser.
* **Audit-Ready:** Every tool features integrated logging and a report export function (📋 Copy Report) to document your analysis.
* **High Performance:** Intelligent chunk-hashing and lazy-loading algorithms ensure speed even with large datasets.

## Quick Start

1.  Clone the repository.
2.  Open any `.html` file directly in your browser.
3.  Upload your files and start analyzing.
